# 💳 Fraud Detection System

## 🚀 Overview
This project predicts fraudulent transactions using Machine Learning.

## 🧠 Features
- Real-time fraud prediction
- User login system
- Dashboard with probability %

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn

## 📊 How it Works
User inputs transaction details → Model predicts fraud probability → Result displayed

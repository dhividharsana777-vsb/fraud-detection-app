# Bank Fraud Detection Using Machine Learning

## Project Overview

Bank Fraud Detection is a Machine Learning based web application that predicts whether a transaction is fraudulent or legitimate.

The system analyzes transaction details and provides quick fraud prediction results using a trained ML model.

---

## Features

- User Registration and Login
- Transaction Fraud Prediction
- Prediction Result Display
- History Tracking
- User Friendly Web Interface

## How to Run

1. Clone the repository
git clone your-repository-link

2. Install requirements
pip install -r requirements.txt

3. Run Flask application
python app.py

4. Open browser
http://127.0.0.1:5000/
